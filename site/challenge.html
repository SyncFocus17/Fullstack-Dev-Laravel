De Developer Challenge

Dit traject test of jij kunt werken als een developer. Niet door alles al te kunnen, maar door te laten zien dat je kunt leren, debuggen, experimenteren en afmaken.

Minimum oplevering (voldoende)

  • ✅ Laravel API met minimaal 2 endpoints (bijv. lijst + aanmaken)
  • ✅ Flutter app met minimaal 2 schermen
  • ✅ Data ophalen vanuit API en tonen in een lijst
  • ✅ Data toevoegen (POST) vanuit app
  • ✅ Basis validatie + nette foutmelding (mag simpel)
  • ✅ Korte demo + pitch (1–2 minuten)

Extra punten (kies jouw strijd)

  • ⭐ Auth (login/register + token)
  • ⭐ Filters / zoeken / sorteren
  • ⭐ Offline caching
  • ⭐ Better UI (consistent, netjes)
  • ⭐ Categories / tags
  • ⭐ Notificatie of deep link

Wat we beoordelen

1) Werkend product
Doet je app wat je belooft? Kun je het laten zien?
2) Developer houding
Bugs aanpakken, proberen, vragen stellen, doorzetten.
3) Progress bewijs
Korte log per week: wat deed je, wat ging stuk, wat leerde je.
4) Extra feature(s)
Je durft iets extra’s te proberen.
