Start hier

Flutter is nieuw. Laravel kennen jullie al. We houden setup simpel en “werkend boven perfect”.

Tools (Windows)

  1. Installeer Flutter SDK
  2. Installeer Android Studio + emulator
  3. Installeer VS Code + Flutter/Dart plugin
  4. Check: flutter doctor

Tip: maak screenshots van je flutter doctor output.

Project structuur

  • Backend: Laravel API (routes, controllers, resources)
  • App: Flutter (screens, services, models)
  • Communicatie: JSON via HTTP

In week 1 mikken we op: één endpoint + één Flutter scherm dat data toont.

Snelle troubleshooting (top 5)

  • Emulator start niet → check virtualization in BIOS + Android Studio Device Manager
  • flutter doctor rood → installeer ontbrekende components
  • API niet bereikbaar → gebruik 10.0.2.2 in emulator voor localhost
  • CORS error → Laravel CORS config / middleware
  • JSON mapping issues → log response en check keys
